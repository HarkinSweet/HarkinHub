# 🎓 HarkinHub

HarkinHub is a modern all-in-one student management portal designed to simplify school life by bringing essential academic and productivity tools into one platform.

Developed by **Akindun Ryan (HarkinSweet)**.

---

## ✨ Features

- 📊 Smart Dashboard
- 📚 Homework Manager
- 📅 Timetable
- 📢 Announcements
- 📖 Digital Library
- 🤖 AI Study Assistant
- 📝 Quiz Centre
- ⏱️ Pomodoro Timer
- 📅 Study Planner
- 💰 Budget Planner
- 🏆 House Points
- ⭐ XP & Achievement System
- 📈 Student Analytics
- 👤 Student Profile
- 👨‍🏫 Teacher Portal
- 🌙 Dark Mode
- 📱 Progressive Web App (PWA)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Chart.js
- Font Awesome
- Progressive Web App (PWA)

---

## 🚀 Getting Started

1. Clone or download this repository.
2. Open the project using **Visual Studio Code**.
3. Launch it with the **Live Server** extension for the best experience.

> **Note:** Some features, including the Progressive Web App and Service Worker, require a local web server and may not function correctly if the HTML files are opened directly.
