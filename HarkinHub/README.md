# 🎓 HarkinHub

HarkinHub is a modern Progressive Web App (PWA) designed to help students manage their academic life through one simple platform.

---

## ✨ Features

### 📊 Dashboard
- Personalized greeting
- Live clock
- Homework statistics
- Progress tracker
- Weekly homework chart
- Study streak
- Motivational quotes
- Notifications
- To-do list
- XP and Level system
- Achievements

### 📚 Homework
- Add homework
- Edit homework
- Delete homework
- Due dates
- Progress tracking

### 📅 Timetable
- Weekly class schedule
- Subject organization

### 📢 Announcements
- School announcements
- Instant updates

### 📖 Library
- Browse books
- Book management

### 🏆 House Points
- House competition tracking

### 🎒 Lost & Found
- Report lost items
- Browse found items

### 💰 Budget Planner
- Weekly allowance
- Expense tracking
- Balance calculation

### 🤖 AI Assistant
- Built-in study assistant
- Homework help
- General questions

### 📝 Quiz System
- Interactive quizzes
- XP rewards
- Score tracking

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Chart.js
- Font Awesome
- Progressive Web App (PWA)

---

## 📱 Works Offline

HarkinHub can be installed as a Progressive Web App and continues working without an internet connection.
